## 4. Key Takeaways & Mistakes to Avoid
- **Avoid Long Multi-Turn Threads:** Long conversation histories drain the token context window. Break technical implementations into fresh chats.
- **Optimize Resource Budgets:** Default to Standard effort for daily operational workflows to save high/max priority limits for complex refactoring.
- **Pasting Structure Correctly:** Provide context framework and relevant technical goals in the very first prompt to eliminate conversational greeting loops.
