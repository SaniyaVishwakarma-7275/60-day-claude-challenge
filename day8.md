# Day 8: Personal Environmental Health Analyzer

## Project Overview
Built a premium, interactive, dark-themed **Environmental Health Analyzer Dashboard** using HTML, Tailwind CSS, and Chart.js. The dashboard focuses on regional air quality (AQI) and hard water metrics, analyzing their specific biometric and dermatological impacts (like skin and hair health).

## What I Learned
* **Data Integration:** Understood how to map environmental data points (AQI, PM2.5, PM10, TDS) to concrete biological risks.
* **Dynamic Charting:** Implemented real-time interactive bar and doughnut charts using Chart.js to compare metrics across different cities.
* **UI/UX Design:** Designed a mobile-responsive dark-theme layout optimized for scannability and professional presentation.
* **Git Workflow:** Practiced creating organized structured directories (`Day8/`) directly through the GitHub repository framework.
